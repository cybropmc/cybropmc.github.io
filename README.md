# cybropmc | Roblox Gameplay Programmer Portfolio 🚀

Welcome to the source code for my personal developer portfolio. Built with a focus on intention, performance, and clean design, this site bridges the aesthetic gap between the Roblox Creator Hub, VS Code, and Apple.

🌐 **Live Site:** [cybropmc.github.io](https://cybropmc.github.io)

---

## 🎨 The Philosophy
I wanted to build one of the cleanest Roblox developer portfolios around. Not by overloading it with flashy animations, but by making it feel intentional. 
* **Zero Bloat:** No heavy frameworks, no copied templates.
* **Premium Feel:** Dark mode by default, clean typography, and subtle interactions.
* **Code-First Design:** A hero section that speaks directly to what I do—writing Luau.

## 🛠️ Tech Stack
Strictly vanilla. Everything is written from scratch specifically for this portfolio.
* **HTML5:** Semantic and accessible structure.
* **CSS3:** Custom properties for theming, dynamic radial mouse glow, and smooth transitions.
* **JavaScript (Vanilla):** Lightweight Intersection Observers for scroll-reveal animations and cursor tracking.

## ✨ Key Features
* **Code Window Hero:** A mock code editor interface displaying my core developer attributes in Luau.
* **Dynamic Mouse Glow:** A subtle, interactive radial gradient that follows the user's cursor.
* **Scroll Reveals:** Content gracefully fades into view as the user navigates down the page.
* **Fully Responsive:** Optimized for desktop, tablet, and mobile viewing.
